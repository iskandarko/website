CryptoCompare Microservice

Microservice collects up-to-date information about cryptocurrency rates from a public API, saves it to the MySQL database and provides it to users in a formatted form. Data is provided regardless of the availability of the API. Developed using JavaScript (Node.js, Express.js).

Source

CryptoCompare микросервис

Микросервис обращается к публичному API за актуальной информацей о курсах криптовалют, сохраняет ее в базу данных MySQL и выдает пользователям в отформатированном виде. Данные предоставляются вне зависимости от доступности API. Разработано на JavaScript (Node.js, Express.js).
